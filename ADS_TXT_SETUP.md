# ads.txt 설정 가이드

## 1단계: Publisher ID 확인

1. 애드센스 대시보드 접속: https://www.google.com/adsense
2. 왼쪽 메뉴에서 **"계정"** 클릭
3. **"계정 정보"**에서 **"게시자 ID"** 확인
   - 형식: `pub-1234567890123456` (16자리 숫자)

## 2단계: ads.txt 파일 수정

1. `public/ads.txt` 파일을 엽니다
2. `pub-XXXXXXXXXXXXXXXXXX` 부분을 본인의 게시자 ID로 교체합니다
   
   예시:
   ```
   google.com, pub-1234567890123456, DIRECT, f08c47fec0942fa0
   ```

## 3단계: 배포 및 확인

1. Vercel에 배포 (또는 Publish)
2. 브라우저에서 확인: `https://your-app.vercel.app/ads.txt`
3. 파일 내용이 표시되면 성공!

## 4단계: 애드센스에서 확인

1. 애드센스 대시보드 > **"사이트"** > **"사이트 관리"**
2. 24시간 후 "Ads.txt 상태"가 **"승인됨"**으로 변경됨
3. 사이트 승인도 1-2주 내에 완료됨

## 주의사항:

- ads.txt 파일은 반드시 **도메인 루트**에 있어야 합니다 (Next.js의 `public` 폴더)
- 파일 이름은 정확히 `ads.txt` (소문자)여야 합니다
- 여러 줄을 추가할 수 있습니다 (다른 광고 네트워크 사용 시)

## 문제 해결:

**"찾을 수 없음" 에러가 계속되면:**
- 배포가 완료되었는지 확인
- URL 직접 접속: `https://your-domain.com/ads.txt`
- 24시간 후에도 인식 안되면 애드센스 고객센터 문의
